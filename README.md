# Banking System

**Description:**

The Banking System is a C# and SQL Server-based application designed to manage banking operations efficiently. It features advanced database operations and supports multiple account types with comprehensive management capabilities for administrators.

**Key Features:**

- **Advanced Database Operations:**
  - Utilizes SQL Server for robust database management.
  - Implements transactions, stored procedures, triggers, and functions for reliable data handling.

- **Multiple Account Types:**
  - **Loan Accounts:** Manage loan repayments and track loan details.
  - **Personal Accounts:** Handle personal transactions and account details.
  - **Admin Accounts:** Comprehensive management of customer information, account details, branch operations, and loan management.

- **Admin Management Capabilities:**
  - Add, edit, and remove customers, accounts, and branches.
  - Monitor and manage transactions, including deposits, withdrawals, and live loan payments.

**Technologies Used:**
- **Programming Language:** C#
- **Database:** SQL Server
- **IDE:** Microsoft Visual Studio
